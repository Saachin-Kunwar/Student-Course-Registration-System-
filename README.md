# 📘 Student Course Registration Form


---

## 📌 Project Title
Student Course Registration Form – A responsive and interactive web-based registration system built using HTML, CSS, and JavaScript with persistent storage.

---

📝 Project Description

This project is a fully functional student registration system where users can submit their personal details, select courses, define skill levels, and provide additional input.

It focuses on real-world form handling, including:

Advanced form validation
Dynamic DOM updates
Persistent data storage using localStorage
User-friendly UI with responsive design

---

🎯 Key Features
📄 Semantic form structure using fieldset and legend
✅ Real-time client-side validation
❌ Inline error handling with focus management
✍️ Live character counter for textarea
🎚 Range slider with dynamic output display
💾 Data persistence using localStorage API
📋 Dynamic registration list rendering
🗑 Delete functionality for each record
📱 Fully responsive layout
♿ Accessibility using ARIA attributes

---
